Este proyecto consiste en un juego clásico de Piedra, Papel o Tijera, desarrollado en Python.
Descripción
El usuario tiene la opcion de elegir: (piedra, papel o tijera) y la computadora genera una elección aleatoria.
El programa determina el ganador según las reglas del juego.
