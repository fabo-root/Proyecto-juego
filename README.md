# 🎮 Juego Piedra, Papel o Tijera

## 📖 Descripción
Aplicación en Python que simula el juego Piedra, Papel o Tijera.  
Permite jugar varias rondas contra la computadora, validar entradas y mostrar el resultado final.

---

## 📂 Archivos del Proyecto

- `proyecto_juego.py` → Archivo principal (menú y control del juego)
- `funciones_juego.py` → Funciones del sistema (validaciones y resultados)

⚠ Ambos archivos deben estar en la misma carpeta.
